👻 Instalación por Termux

<details>
<summary><b>👑 Haga click para ver los comandos</b></summary>Instalación manual por Termux

termux-setup-storage

apt update && apt upgrade && pkg install -y git nodejs ffmpeg imagemagick yarn

git clone https://github.com/thecarlos19/black-clover-MD && cd black-clover-MD

yarn install && npm install

npm start

> ⚠️ Si después de poner el número de WhatsApp salen letras en rojo, no se preocupe. Es normal.
